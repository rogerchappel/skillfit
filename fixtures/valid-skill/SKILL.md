# valid-skill

Use this skill when an agent needs to convert a repeatable workflow into a reviewed instruction artifact.

## Inputs

A task brief, local files, and examples showing when the workflow should and should not activate.

## Side Effects

The skill does not write to external systems without approval and clearly separates dry-run review from live actions.

## Examples

1. Draft a skill proposal from a runbook.
2. Validate a skill before sharing it.

## Verification

Run tests, smoke commands, and fixture checks before approval.
